# Jeu de nim

Le jeu commence avec **7 allumettes**. À chaque tour, un joueur
retire entre **1 et 4 allumettes**. Le joueur qui retire la dernière
allumette perd la partie.

## Fonctionnalités

- Mode deux jouer ou contre l'ordinateur

## Structure du projet

```text
nim_game/
├── Node.py       # Classe représentant un nœud de l’arbre de jeu
├── game.py       # Gestion des joueurs et des saisies
├── minimax.py    # Création de l’arbre et algorithme Minimax
├── nim_game.py   # Programme principal
├── player.py     # Classe représentant un joueur
└── README.md
```
